# 工作流设计

## 总流程

用户问题
→ 意图识别
→ 低敏信息补全
→ 动态检索或平台知识库查询
→ 大模型结构化生成
→ 合规校验
→ 输出下一步动作

## 意图分类

| 意图 | 识别关键词 | 输出重点 |
| --- | --- | --- |
| policy_query | 政策、补贴、贴息、专精特新、创业担保贷款、申报 | 政策方向、入口、材料、官方核验 |
| financing_prepare | 融资、贷款、周转、采购、开店、资金 | 准备清单、路径说明、风险提醒 |
| document_draft | 帮我写、生成、说明、申请书、材料 | 可复制初稿、待确认字段 |
| compliance_check | 安全吗、靠谱吗、风险、合同、宣传、费用 | 风险等级、问题点、替代表达 |
| operation_advice | 经营、现金流、库存、客户、复购 | 通用经营动作、复盘模板 |
| case_reference | 案例、公开实践、普惠金融、AI 风控、联合会 | 案例拆解、可借鉴点、边界 |
| publish_asset | 智能体、发布、广场、标签、简介、首屏问题 | 发布文案、关键词、审核风险 |

## 各模块输出模板

### 政策找一找

1. 可关注政策方向
2. 初步适用条件
3. 官方核验入口
4. 可能需要的材料
5. 申报前注意事项

### 融资准备助手

1. 你的需求属于什么场景
2. 可了解的融资路径
3. 优先准备的材料
4. 不建议做的事
5. 下一步 3 个动作

### 材料生成工具

1. 文档初稿
2. 【待确认】字段
3. 提交前检查清单
4. 合规提示

### 合规避坑问答

1. 风险等级：低 / 中 / 高
2. 触发风险点
3. 为什么要谨慎
4. 安全替代表达或动作
5. 建议核验渠道

### 小微经营知识库

1. 现状判断
2. 可执行动作
3. 简易表格或模板
4. 每周复盘方式
5. 边界提醒

### 服务案例与公开实践

1. 案例背景
2. 服务动作
3. 可借鉴点
4. 风险边界
5. 对外表达建议

## 动态检索策略

如果平台支持联网搜索：

1. 政策类问题优先搜索官方来源。
2. 最新政策、地区政策、数据口径必须检索后回答。
3. 回答中标注“建议以官方页面为准”。
4. 不要引用来源不明的贷款中介、营销软文。

如果平台不支持联网搜索：

1. 不输出“最新”“当前”等确定性判断。
2. 给政策方向和核验路径。
3. 引导用户到当地政务服务网或主管部门核验。
