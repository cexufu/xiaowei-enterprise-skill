# 小微企业服务助手 Skill 包

这是一个平台无关的 Skill / 智能体能力包，用于发布到 Coze、文心智能体、Dify、FastGPT、豆包智能体等平台。

核心原则：不依赖自建 Node 服务，不依赖 Cloudflare，不在前端暴露 API Key。优先使用平台原生大模型、联网搜索、知识库、工作流和审核机制。

## Skill 定位

名称建议：小微企业服务助手

一句话：帮小微企业把政策、融资准备、材料和合规风险先理清楚。

发布主体：中小企业联合会

案例原则：如需案例，只引用经过审核的公开实践，且案例只是补充说明，不是产品主叙事。

## 能力模块

1. 政策找一找
2. 融资准备助手
3. 材料生成工具
4. 合规避坑问答
5. 小微经营知识库
6. 服务案例与公开实践
7. 智能体发布资产包

## 文件说明

- `universal_prompt.md`：所有平台都可直接使用的核心系统 Prompt。
- `workflow_spec.md`：工作流与意图识别设计。
- `platform_mapping.md`：Coze、文心智能体、Dify/FastGPT 等平台字段映射。
- `publish_copy.md`：名称、简介、标签、开场白、首屏问题。
- `guardrails.md`：合规边界、拒答规则、风险提示话术。
- `test_cases.md`：上线前测试用例。
- `skill_manifest.json`：平台无关结构化配置。

## 推荐上线方式

第一阶段不要做复杂插件和代码节点，先做「纯 Prompt + 工作流 + 联网搜索 + 少量知识素材」。

第二阶段再接官方政策检索、联合会服务入口、案例库和表单收集。

第三阶段再考虑 API 化、CRM 线索、数据看板和跨平台统一管理。
